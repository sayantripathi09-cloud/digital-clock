CLOCK FEATURES

BUTTON FUNCTION

B1 (SET Button)
  • Single press → Enter set mode / Increment Hour / Minute / Date / Month
  • Long press → Move to next field / Last long press saves Time and Date to RTC and starts clock
B2 (BRIGHT Button)
  • Single press → Change brightness (5 levels)
  • Long press → Clock RUN / PAUSE toggle

TIME & DATE SET MODE

  • SET mode: Digit Red colour
  • Selected field: Yellow colour
  • Colon steady ON (not blinking)

AUTO COLOUR CHANGING DISPLAY

  • Running mode: Colour auto change
  • Total 60 colours
  • If you change the time, colour may change in code

SMART BRIGHTNESS SYSTEM

  • Auto Brightness (RTC time):
    • 6 a.m – 6 p.m → Day brightness 50
    • 6 p.m – 6 a.m → Night brightness 25
  • Manual Brightness Override: Press Brightness button → Auto brightness off

AUTO DATE DISPLAY

  • Normal time: 6 sec
  • Date display: 2 sec

BUZZER FEEDBACK SYSTEM

  • Power ON: Double beep
  • Button short press → High beep
  • Long press → Low beep
  • Per hour beep

RUN / PAUSE MODE

  • Clock pause: Time freeze
  • Display White colour
  • RTC continues running in background

If you want to make any changes, you will need to modify the code. Example: Colour, Time & Date display duration, Colon blink time, etc.

COMPONENT LIST

  • Microcontroller: Arduino Nano (or similar)
  • RTC Module: PCF8563 (or similar)
  • Passive buzzer
  • WS2812B LED × 30
  • 5V Power supply
  • Buttons × 2
